# Employee Attrition Analysis using Python

## 📌 Project Overview

Employee attrition is a major challenge for organizations as it directly impacts productivity, recruitment costs, and employee morale.

This project performs an end-to-end HR analytics workflow using Python to analyze employee data, identify the major factors influencing employee turnover, and generate actionable insights that can help organizations improve employee retention.

The analysis includes data cleaning, exploratory data analysis (EDA), visualization, and business recommendations.

---

## 🎯 Project Objectives

- Analyze employee attrition patterns
- Identify the factors responsible for employee turnover
- Discover trends across departments, job roles, salary levels, age groups, and work-life balance
- Provide data-driven recommendations for improving employee retention

## 📈 Key Insights

Some important insights obtained from the analysis include:

- Employees working overtime are more likely to leave.
- Lower monthly income is associated with higher attrition.
- Certain departments experience significantly higher employee turnover.
- Younger employees tend to leave more frequently than experienced employees.
- Employees with poor work-life balance show a higher attrition rate.

---

## 💡 Business Recommendations

- Improve work-life balance initiatives.
- Reduce excessive overtime.
- Review salary structures for high-risk employee groups.
- Increase employee engagement and recognition programs.
- Provide clear career growth opportunities.
- Focus retention efforts on departments with high attrition.
